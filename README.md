## 🚀 Project Overview

EduPlan is a full-stack web application designed to streamline the student experience. It features a responsive frontend, a secure REST API, and an integrated AI chatbot for personalized assistance.

### 🏗️ Architecture (Monorepo)

We use a **Monorepo** structure to keep all parts of the project unified:

* **📂 /frontend**: The user interface (React, Vite, Tailwind CSS).
* **📂 /backend**: The API server (Node.js, Express, MongoDB).
* **📂 /ai_model**: The intelligence layer (Python, AI/Chatbot modules).
